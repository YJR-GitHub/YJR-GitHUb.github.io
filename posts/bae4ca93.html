当然这是对小白比较友好的一款多环境自动配置的面板,还有宝塔面板也是.

PHPStudy（小皮）V8.1 中使用cmd进入MySQL命令行模式

  1. 找到phpstudy_pro中的mysql可执行文件所在的位置(安装位置)

  2. 输入命令mysql -u root -p

  3. 然后输入密码
    （注意这里的root是phpstudy里面root数据库的用户名）

    成功进入cmd模式是下面的界面：

  4. 未成功进入报错解决方案（如果成功进入咋忽略以下步骤）
    ERROR 1045 (28000): Access denied for user 'ODBC'@'localhost' (using password: NO)
    ERROR 1045 (28000): Access denied for user 'ODBC'@'localhost' (using password: YES)
    windows下，以上两个错误的解决方法

  5. 找到配置文件my.ini ，然后将其打开，可以选择用NodePad++打开

  6. 打开后，搜索mysqld关键字找到后，在mysqld下面添加skip-grant-tables（此操作的意思是是不需要密码进行进入），保存退出。

  7. 把你的phpstudy进行重新启动

  8. 然后运行cmd，输入mysql -u root -p就可以不用密码登录了，出现password：的时候直接回车可以进入。

  9. 进入mysql数据库：
    mysql> use mysql;Database changed

  10. 给root用户设置新密码：
    mysql> update user set authentication_string=password('123456') where User="root";

    提示：Query OK, 1 rows affected (0.04 sec)Rows matched: 1 Changed: 1 Warnings: 0

  11. 刷新数据库mysql> flush privileges;

    提示：Query OK, 0 rows affected (0.01 sec)

  12. 退出mysql：mysql> exit

    提示：Bye

    PS：123456为新密码，用户可根据自己需要修改成自己的密码

  13. 改好之后，再修改一下my.ini这个文件，把我们刚才加入的"skip-grant-tables"这行删除，保存退出再重启phpstudy就可以了。
